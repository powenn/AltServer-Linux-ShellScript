Usage: [OPTION]

OPTIONS

  1, --Install AltStore

    Install ipa in Folder 'ipa',make sure you have put ipa files in the Folder before run this
  d, --Restart Daemon mode
    Restart Daemon mode to refresh apps or AltStore
  e, --exit
    Exit script
  h, --help
    Show this message
